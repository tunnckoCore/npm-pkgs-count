> Count npm user packages from npmjs.com website profile. API and CLI.

## Install
```
npm i --save npm-pkgs-count
npm test
```


## Usage
> For more use-cases see the [tests](./test.js)

```js
var npmPkgsCount = require('npm-pkgs-count');
```
